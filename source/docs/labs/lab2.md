## Отчёт

### Описание задачи

Требовалось написать набор функций для работы с NumPy: создание и преобразование массивов, арифметические операции, матричные вычисления (умножение, определитель, обратная матрица, решение СЛАУ), статистический анализ данных оценок студентов и построение графиков.
